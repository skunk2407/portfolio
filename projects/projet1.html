TP Fleuriste

Type de projet : Base de données

Contexte : Ce projet a été réalisé dans le cadre de ma formation en BTS SIO, option SLAM, à l'Aftec de Rennes. L'objectif était de concevoir une base de données pour gérer les informations d'un magasin de fleurs.

Objectifs du projet

  • Créer une base de données pour gérer les produits, commandes et clients.
  • Utiliser SQL et MySQL pour implémenter les fonctionnalités de gestion des données.

Développement et réalisation

La base de données a été conçue avec plusieurs tables interconnectées, incluant des relations entre les produits, les commandes et les clients. Des requêtes SQL ont été utilisées pour manipuler les données efficacement.

Quelques fonctionnalités clés du projet :

  • Création des tables et relations.
  • Écriture de requêtes SQL pour gérer les commandes et produits.
  • Résolution des problèmes de normalisation des données.

Challenges rencontrés

Un des défis majeurs était la gestion des relations entre les différentes tables pour garantir la cohérence des données. Des erreurs dans les requêtes SQL ont été corrigées grâce aux outils de débogage.

Conclusion

Ce projet m'a permis de consolider mes connaissances en gestion de bases de données et en SQL. J'ai appris à structurer des bases de données complexes et à résoudre des problèmes liés à la normalisation et aux relations entre les tables.
